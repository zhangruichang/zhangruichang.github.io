NQuenes

8月 2 2014

这道题目剪枝和不剪枝，性能天壤之别，如果每次都是到头了再判断是否解合法，N=8都要十几分钟，而如果在每一次想下层递归，都判断当前
摆放的皇后是否合法，N=10都只需要5s，因为这个是指数爆炸的搜索空间O(N^N)，空间，如果到最后解的话非常慢，而每次都会提前删减掉
很多无用解，减少很多次递归。

    vector<vector<string> > solveNQueens(int n) {
        qpos.clear();
        board.clear();
        solution.clear();
        string eachrow="";
        for(int i=0;i<n;i++)
            eachrow+='.';
        for(int i=0;i<n;i++)
            board.push_back(eachrow);

        dfs(0,n);
        return solution;
    }
