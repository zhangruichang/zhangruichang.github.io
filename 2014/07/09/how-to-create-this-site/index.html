how to create this site

7月 9 2014

之前在各路大神的博客里乱逛，无意中问到怎么搭建博客，结果大神慷慨的发给我发了个链接。
而且之前在知乎也看到，某大神说，程序员要有一个自己的主页，于是我打算倒腾一下，hexo的作者说这个比ocpress wordpress都要好的多，
于是我打算搞一搞。

搭建的时候，第一个问题就是git居然不识别npm，我怎么都弄不明白，windows cmd也可以的，后来终于打印path发现问题，好像是git的shell不能跨盘符，
我在H目录 git结果path里C盘一个路径就识别不出来，于是换了一个目录。还是说最后要加个;, 大概就是这几种可能。

然后后面步骤一步一步都比较顺利，最后localhost是理解就可以了，但是挂到github要等<=10min, 然后顺便搞搞个性化配置。

发表新文章

hexo new “title”

然后编辑本地source_posts目录下的对应.md文件，但是里面是用markdown语法，菜鸟还没学过，
不过这个规则很松，随便写文本都可以，只是要有特效可能要有markdown语法吧。。。

同步到github
hexo generate //类似于本地同步一样
hexo deploy//同步到github

同步到github就是每次都要登陆用户名密码，找找看可以不可先配置在一个文件，然后直接automatic，记得之前看过一个blog，好像设置path啥的

然后就可以了，但是有时候稍微会慢几秒到几十秒，或者是brownser缓存的问题。。。

如果本地localhost: 4000浏览就需要
hexo server

类似于tomcat bat, 让他一直挂着，不过一般都不用本地啦。

接下来打算先优化一下博客显示，最主要是买个域名，然后有个自己的个性域名，就done了！
